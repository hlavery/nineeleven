    Credits

    This website was created using HUGO, a static website generator which processes Markdown text into HTML, avoiding the complexities of hypertext.

    Hugo requires a “theme,” and I chose DocDock. DocDock is usually used for software documentation, but I found it very effective for organizing scanned letters and photos.

    The final build is done on Fleek. I have been editing the Markdown text files using the venerable (ca. 1999) Linux terminal editor, nano.

    For any questions, please contact me, Hugh Potter Lavery, at hlavery@comcast.net.
